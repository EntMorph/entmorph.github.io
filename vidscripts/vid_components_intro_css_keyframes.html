3. Add keyframes for border-radius

Top

  • We'll start by adding an animation,
  • Goto > Code > :styles
  • click on Collapse All
  • click on CSS Style Rules > Add
    • which opens up a sub-menu;
  • click on @Keyframes
    • to add a Keyframes Rule editor
  • It has an Id,
    • which we'll change to radii-animation
  • and a :frames section
    • which lets us edit the actual keyframes of our animation;
  • [open (:frames) ]
  • There are two keyframes by default
    • one at the start (0%)
    • and the other at the very end (100%) of the animation
  • Let's add one more frame:
    • click on Add Frame > Empty to add an empty keyframe
    • notice that now the frame stops have been re-arranged to 0%, 50% and 100%;
    • this happens automatically, to save you from having to manually set the stops yourself;

5. Add .animate-radii Style Rule

Top

  • Goto > :styles
  • click on Add > Style Rule
    • Selector: .animate-radii
    • properties
{:animation "radii-animation 5s linear infinite"}

6. Add .pause style

  • Goto > :styles
  • click on Add > Style Rule
    • Selector: .pause
    • properties
{:animation-play-state "paused"}

9. Add checkboxes to :render

Top

  • Now we can add our checkboxes to trigger and pause the animation;
    • We'll add two new rows to the table;
    • Goto > :render
    • [scroll-to table > tbody]
    • [open (tbody) ]
    • TWICE click on Add :tr
  • We'll update the first new row
[:tr
 [:td "animate?"]
 [:td [:input/checkbox
       {:value @animate?}]]]
  • Then the next row
[:tr
 [:td "paused?"]
 [:td [:input/checkbox
       {:value @paused?}]]]

11. Add @animate? to svg :class map

Top

  • [scroll-to svg element]
  • [open (svg element) ]
  • [open (Attributes) ]
  • click on Add
  • Change Editor from String to Map
  • click on Add Entry
    • Key: "animate-radii" (the name of the class we want to trigger)
    • Val: @Reactive > @animate?
  • This is the first time we're seeing a map as the value of a class attribute;
    • what it does is to add the class to the element only when the @animate? state variable is true
    • And the only way to set @animate? to true is by checking the checkbox
  • So let's test this
