18.1. Feature: Renaming an Id

Top

  • This "automatic reference updating" is a general feature of entmorph editors;
    • if you rename an Id,
      • in the place where you define it;
    • then all the references to that Id will automatically be updated to show the renamed Id;
  • This works for all Ids, not just Input Parameters;
  • The great thing about this is that you never have to perform a Search and Replace like you would in a text editor,
    • which should save you a lot of time,
      • as well as reduce errors;

19.1. Useful features we covered in this video

  • From just what we've covered in this video,
    • I hope you're starting to gain a better understanding of how entmorph components can transform the way you develop web applications;
  • First, entmorph provides a browser-based coding environment:
    • so no downloads,
    • no installations,
    • just sign-up,
    • and start creating;
  • entmorph components provide a unified build and preview environment,
    • so there's no need to switch between your code editor and the browser;
  • The entmorph code editor contains sub-editors,
    • which provide commands that allow you to add only valid items,
      • often selecting them from drop-down menus,
      • or textboxes which limit what you can type;
    • this makes it very difficult to make most types of mistake, especially compared with text-based coding;
  • The entmorph component :render editor provides a visual development experience which should dramatically accelerate your User Interface development process compared with traditional HTML coding;
  • and finally, automatic reference updating when you rename an Id,
    • should eliminate error-prone search-and-replace operations
