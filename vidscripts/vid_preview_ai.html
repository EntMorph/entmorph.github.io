IF WORKING:

  • I'll first click on the right-string;
    • and a Properties panel appears;
    • containing Attributes and Styles for this object;
  • If I scroll all the way down, underneath Styles there's a Constraints panel,
    • which contains N constraints;
    • a constraint creates a relationship between an object and another object,
    • or an object and itself;
  • In this case we have a :self-attr and a :constant-minus-attr;
  • :self-attr
    • to.x = from.x
      • the start and end of the string should always be at the same x-position;
      • in other words,
        • "the line should always be vertical"
  • :constant-minus-attr (constant minus attribute constraint)
    • right-string.to.y = 550 - left-string.to.y
      • the y-position of the end of the right string is 550 minus the y-position of the end of the left string;
      • this ensures that pulling the left string down will pull the right string up,
        • and vice versa;

9. Conclusion

  • In this video, we've used Ai to create:
    • two functions, and
    • four components
      • and in fact each component can be thought of as an app,
        • with the added advantage that you can include a component in another component,
        • which of course you can't do with apps;
  • The functions and their tests were generated entirely by the Ai,
    • and Entmorph provided controls for editing and testing them;
  • But all the components were created using Entmorph's existing capabilities;
    • all the Ai did was to take the user prompt and translate it into a specification which Entmorph then used to generate the components;
  • This approach provides us with tremendous advantages over just asking Ai to generate everything for us;
    • For example, I've seen people submitting between hundreds and low thousands of prompts trying to generate apps because an Ai prompt was the main or only tool available;
  • With Entmorph, you can use the Ai for the initial generation, and maybe update it using Ai a small handful of times if you need to,
    • but in many cases you should be able to tweak it using any of the visual editors available, some of which we have seen in this video,
    • without having to edit a line of code;
  • And since Entmorph uses its own capabilities to generate components,
    • this greatly increases the chances of generating robust applications since it involves using components and other features which have been thoroughly tested before being released to the public;
    • this is a much more reliable approach than asking an Ai to generate all the code itself, which usually results in hallucinations which can be very difficult to fix, and the code itself is very difficult to maintain;
    • this also saves you from having to submit prompts just to fix bugs which the Ai itself introduced;
  • I hope you've enjoyed this preview of Entmorph's Ai capabilities;
    • by the time you watch this, these should have been released, or something even better should be in place;
  • Please check out Entmorph.com and the Youtube channel youtube.com/@entmorph,
  • and of course please share with anyone who might find this video interesting.
