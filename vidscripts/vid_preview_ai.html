  • Now, if I click on "Run Function Test",
    • we see the Test Result is the same as the Expected value,

  • scroll-to Function Call
    • The first argument canvas-context is the context of the Canvas element below,
      • and has been provided for us by the test;
    • The second argument is a map of values, which we can play with;
  • The first thing I'm going to do is to click on Live Edit,
    • so that any change I make will update the canvas immediately, rather than having to press "Run Function Test" each time I make a change;
  • What I'll now do is open the map,
    • open the editor for :n,
    • and move :n to the very bottom so that we can see the canvas at the same time;
  • I'll open the editor for :n

  • Now when I update n,

  • We can of course adjust any of the values to see if our function works as expected;

Conclusion

  • So we can generate a function which draws onto a canvas,
    • and test it immediately in the browser,
    • whether the Ai generates the function for us, or we create it ourselves;
